答:工程化可以解决工作中出现的重复工作，例如部署，引入常用库等。
  1.自定义模板，可以减少创建一系列文件所带来的的工作。
  2.自定义自动化构建项目，可以将自己在工作中积累的代码直接引入到项目中，不必多次重复的复制粘贴，提高代码的复用率。
  3.通过工程化的项目，可以大幅度减少项目中带来的重复操作。

答:给项目带来了工程化的基因,减少开发人员的工作量，并且提供给开放人员更多的自由度。
